Resource with and without library

Div with Library

This div has a background image specified using an EL expression in the stylesheet.

The expression is structured to contain a library name along with the resource id:

resource['css:images/background.png']

Div without Library

This div has a background image specified using an EL expression in the stylesheet.

The expression is structured so that it only includes the resource id (no library name).

resource['css/images/background.png']
